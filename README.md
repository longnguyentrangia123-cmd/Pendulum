# Pendulum

A functional python code that shows the movements of a pendulum in perfect conditions (without external forces).

![image alt](https://github.com/longnguyentrangia123-cmd/Pendulum/blob/4ab3a8fe77fe889c7d9c783d415223a03923fa28/state%20diagram.png)

To use this code, copy and paste the code onto a code editor, for example, Visual Studio, and run the code.

If any bugs or errors have been found, please add a comment next to the code that causes the error/bug or add a comment at the end of the code stating the error message.
